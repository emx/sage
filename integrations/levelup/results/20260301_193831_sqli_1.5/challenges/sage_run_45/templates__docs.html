Telemetry Endpoint
POST /api/v1/telemetry/metrics

Allows supervisors to query vessel status directly from the telemetry layer.

Payload Format:
{
  "filter": "vessel_id = [ID]"
}
Response:
{
  "status": "active|inactive",
  "node": "PRIMARY_VML"
}

Note: All input is sanitized via the VML-WAF module. Direct SQL access to vanguard_system_meta or users is strictly prohibited.
