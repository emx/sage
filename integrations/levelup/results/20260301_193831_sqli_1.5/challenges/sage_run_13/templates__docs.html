Logistics API Documentation (v1.4)

This API is intended for use by the KestrelMobile logistics application.

POST /api/v1/logistics/search

Searches for propulsion units assigned to a specific facility.

Request Body:
{
  "facility_id": "string"  // Example: "FAC-01"
}
Response Body:
{
  "status": "success",
  "results": [
    { "serial_no": "VK-100", "model": "Kestrel-9", "status": "Active" }
  ]
}
Security Note: All POST requests are inspected by the VK-Shield WAF. Payloads exceeding standard schema parameters will be logged and blocked.
