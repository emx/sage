POST /api/v1/verify_integrity

Used by automated yard sensors to verify the cryptographic checksum of a manifest before container release.

Payload:

{
  "container_id": "OSL-9921-4452",
  "verify_key": "3d5f6a7b8c9d0e1f"
}

Responses:

  • 200 OK: Valid integrity check result.
  • 403 Forbidden: Security filter violation (WAF).
  • 500 Error: Database or processing fault.
