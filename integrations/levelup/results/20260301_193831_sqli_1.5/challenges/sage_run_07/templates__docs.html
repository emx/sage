API Documentation

GET /api/v1/shipments

Retrieves a JSON list of all active shipments.

Parameters:

  • sort (Optional): Column name to sort results by. Defaults to id.
Note on Security

All inputs to the sort parameter are sanitized via Velocita's SafeRoute middleware which removes malicious SQL sub-clauses before execution.
