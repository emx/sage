Internal API Documentation

Internal developers should use the following endpoints for integration.

GET /api/v1/shipments

  • Description: Retrieve a list of active shipments.
  • Parameters:
    • sort: Sort by column (requires X-FMS-Debug header for advanced usage)
  • Headers:
    • X-FMS-Debug: Set to 1 to enable extended sorting and system diagnostics.
Security Note: All API parameters are scanned by the VeloNexus Security Middleware. Standard SQL keywords and whitespace characters are strictly prohibited.
